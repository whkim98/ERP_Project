# <img src="https://github.com/whkim98/ERP_Project/assets/86636344/4e1233be-7b50-4619-bc70-f13dd75c71bd" width="30"/>
<b>물류, 인사, 재무 등 다양한 기능을 한 번에!</b> 쉽고 간편하게 전사적 자원 관리를 도와줍니다.<br/>
다양한 기능으로 더욱 효율적으로 업무 능력을 높여줍니다!<br/><br/>

## 주요기능
* 인사(근태, 정산, 휴가 등) 관리
* 거래처 물자 및 물류 관리
* 재무제표 등록, 수정
<br/><br/>
